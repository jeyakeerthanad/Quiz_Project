#  CS Quiz Game (Python)

A simple and interactive **Computer Science Quiz Game** built using Python. Tests your knowledge on core CS topics like Data Structures, Algorithms, Python, and Git.

---

##  Features

* 5 Computer Science questions
* Multiple choice format (1/2/3/4)
* Questions shuffled randomly every time
* Instant right/wrong feedback after each answer
* Final score with performance message
* Personalized with your name

---

##  Technologies Used

* Python
* `random` module (for shuffling questions)

---

##  How to Run

1. Make sure Python is installed on your system
2. Clone this repository:

```bash
git clone https://github.com/jeyakeerthanad/Quiz_Project.git
```

3. Navigate into the folder:

```bash
cd Quiz_Project
```

4. Run the file:

```bash
python quiz.py
```


---

##  How It Works

* User enters their name
* 5 CS questions are shuffled and displayed one by one
* User types the number of their answer
* Correct or wrong feedback is shown instantly
* Final score and performance message displayed at the end
